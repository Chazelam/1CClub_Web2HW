Задание: Создание игры "Камень, ножницы, бумага"

Описание: Создайте игру "Камень, ножницы, бумага" с использованием HTML, CSS и JavaScript.

Требования:

  1. Создайте интерфейс с тремя кнопками: "Камень", "Ножницы", "Бумага".
  2. Реализуйте логику игры:
    • Компьютер случайным образом выбирает один из вариантов.
    • Сравните выбор игрока и компьютера, определите победителя.
    • Покажите результат на странице.
  3. Оформите игру с помощью CSS (добавьте иконки или изображения для кнопок).
  4. Разместите проект на GitHub Pages.

Дополнительные задачи:

  1. Добавьте счетчик побед игрока и компьютера.
  2. Реализуйте возможность играть несколько раундов подряд.
  3. Добавьте звуковые эффекты для действий игрока и результатов.
