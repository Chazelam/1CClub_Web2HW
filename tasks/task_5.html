Задание: Создание формы регистрации

Создайте форму регистрации с полями "Имя", "Email" и "Пароль". Используйте тег

.

Требования:

  1. Поле "Имя" не должно быть пустым.
  2. Поле "Email" должно содержать символ "@".
  3. Поле "Пароль" должно быть не короче 6 символов.
  4. Оформите форму с помощью CSS (цвета, шрифты, отступы).
  5. Выводите сообщения об ошибках рядом с полями, если валидация не прошла.
  6. При успешной валидации показывайте сообщение об успешной регистрации (например, через alert).
  7. Отмените стандартное поведение формы (отправку данных на сервер) с помощью event.preventDefault().

Дополнительные задачи:

  • Добавьте анимацию для кнопки "Зарегистрироваться" (например, изменение цвета при наведении).
  • Сделайте так, чтобы сообщения об ошибках исчезали, когда пользователь начинает исправлять поле.
  • Добавьте поле "Подтверждение пароля" и проверьте, совпадает ли оно с полем "Пароль".
